# Success Club 2026 Scholarship Portal

A scholarship website for the **Success Club 2026 - SDG 4: Quality Education** project.

The website helps underprivileged students learn about scholarship support, prepare their application, and submit the official application form. It is designed to be clear, simple, and easy to use on both desktop and mobile.

## Website Pages

### Home

The homepage introduces the mission of the scholarship portal: helping underprivileged students continue their education. It includes a large hero image, a short mission statement, and scrollable sections explaining the importance of access, dignity, opportunity, fair review, and quality education.

### My Applications

The My Applications page is the main student action page. It includes:

- Eligibility checker
- Application checklist
- Application timeline
- FAQ section
- Button to open the official application form
- Pop-up reminder before leaving the website for the form

Students use this page to prepare their details and submit their application through the official Google Form.

### Contact Us

The contact page gives students a way to reach the scholarship team for help. Students are asked to use the same email address they used in the application form so their response can be found quickly.

### Admin Page

The admin page is hidden from the public menu. It is used by the review team to access submitted application responses and manage review notes/status privately.

## Student Flow

```txt
Visit Home -> Open My Applications -> Check eligibility -> Prepare details -> Submit form -> Wait for review
```
